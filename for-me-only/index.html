    Other components ðŸ”—

    • Mark: ==Text==.
    • Keyboard: [[Ctrl]].
    • Reference: {% ref "url" %} (mush have "").
    • Inline text color: {color:red}text{color} (without space)
    • {:.noindent} before a list, not indent a list.
    • {:.indent} before a list, indent a list.
    • {:target="_blank"} after an url.
